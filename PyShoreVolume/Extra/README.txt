
NATIONAL NETWORK OF REGIONAL COASTAL MONITORING PROGRAMMES - WAVE DATA USE

1. FILE FORMATS

a) Quality-controlled (QC) data formats: Xxx_wavesYYYY, Xxx_wavesYYYYMM
	- These data have undergone full quality control. Poor data is flagged, not removed.
	- Full details of the flagging and quality control procedures can be found in the Quality Assurance and Quality Control of Wave Data handbook. https://coastalmonitoring.org/ccoresources/dataqualitycontrol/
	- A description of each of the wave parameters provided is available in the Wave Parameter Handbook on the Resources Page. https://coastalmonitoring.org/ccoresources/waveparameterhandbook/

b) 'All wave parameter' file formats: Xxx_allwaveparametersYYYY, Xxx_allwaveparametersYYYYMM
	- Be aware that these data have not undergone any quality control. For standard engineering practise please use the full quality-controlled data.
	- These wave files contain the full range of parameters available from a Datawell DWR MkIII wave buoy and an additional two wave parameters Energy Period (Te) and Wave Power (P).
	- A description of each of the wave parameters provided is available in the Wave Parameter Handbook on the Resources Page. https://coastalmonitoring.org/ccoresources/waveparameterhandbook/
